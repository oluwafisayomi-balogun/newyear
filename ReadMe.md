# New Year Countdown

A clean, festive countdown website that ticks down to midnight on New Year's Day. 

![Preview](favicon.svg)

---

## Features

- **Live countdown** — Days, Hours, Minutes, Seconds with a smooth bump animation on every tick
- **Timezone switcher** — 25 timezones supported, auto-detects yours on load
- **Celebration screen** — triggers at midnight with bouncing emojis, a welcome message, and a confetti shower
- **Floating headline** — "NEW YEAR" gently bobs up and down
- **Monochromatic orange palette** — warm, dark, and cohesive
- **SVG favicon** — clock icon
